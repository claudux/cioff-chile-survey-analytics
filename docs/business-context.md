# Contexto de negocio

CIOFF Chile requería una forma más estructurada de recopilar información sobre la experiencia de asistentes y participantes, con el fin de detectar oportunidades de mejora y fortalecer la organización de futuras actividades.

A partir de esta necesidad, se desarrolló un flujo completo de levantamiento, captura, extracción, limpieza y análisis de datos orientado a generar hallazgos accionables.
