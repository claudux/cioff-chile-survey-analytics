# Proceso de encuesta y captura

El proyecto incluyó la construcción del instrumento de encuesta, la automatización de recepción de respuestas y la preparación de la base de datos para su análisis posterior.

## Etapas principales
- diseño de encuesta
- captura automatizada de respuestas
- extracción de datos
- limpieza y validación
- consolidación de información
- análisis y visualización
